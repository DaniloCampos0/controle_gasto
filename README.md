# Controle de Gastos

Sistema em Python para controle financeiro pessoal com persistência de dados e análise básica.

## Funcionalidades
- Adicionar gastos
- Editar e remover gastos
- Exportação CSV
- Análise por categoria
- Média de gastos
- Filtro por mês

## Tecnologias
- Python
- JSON
- CSV

## Objetivo
Praticar lógica de programação e análise de dados.
